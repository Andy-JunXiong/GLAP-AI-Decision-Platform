# GLAP Port Disruption Decision Brief

An interactive product demo showing how GLAP turns port congestion, labour-strike probability, FCL storage exposure, and inventory cover into a traceable diversion decision.

## Run locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm test
npm run lint
```

The scenario is synthetic. Approval and rejection controls update the local demo state and do not send instructions to a carrier, terminal, or transport-management system.
